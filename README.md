# Skill-Bridge Career Navigator

## Project Overview
An intelligent career skill gap analyzer and personalized learning roadmap generator built with Streamlit and LLaMA 3.1 AI (via Groq).
Paste your resume or enter your skills → select a target role → get instant skill gap analysis with a personalized learning roadmap.

This tool aims to solve the persistent challenge candidates face when trying to understand the gap between their current skills and the specific requirements of their target role. 

## Features
* **🧠 AI Skill Extraction**: LLaMA 3.1 via Groq extracts skills from unstructured resume text.
* **🔄 Smart Fallback**: Reliable rule-based keyword matching seamlessly takes over when AI is unavailable.
* **🎯 Skill Gap Analysis**: Intelligent skill gap matching featuring priority levels and matched vs. missing skills breakdown.
* **🔴🟡🟢 Priority Levels**: Classifies missing skills into High, Medium, and Low importance.
* **🗺️ Learning Roadmap**: Custom week-by-week learning steps for missing skills generated by AI or using precise fallback data.
* **🔍 Search & Filters**: Search fields for roles/skills and priority-based filters.
* **📊 Readiness Summary**: Provides a quick “You are X% ready” evaluation along with experience level detection.
* **🧪 Demo Mode**: A toggle to simulate AI failure and test the fallback workflow.
* **✅ Unit Tests**: Comprehensive test suite covering both the AI generation and rule-based fallback integrations.

## System Flow
1. **Input**: A candidate pastes their resume text or inputs skills manually, then selects a target role.
2. **Extraction**: The system calls LLaMA 3.1 via the Groq API to extract the technical skills, applying whitelist filtering to eliminate noise. Alternatively, a regex fallback is triggered if the AI fails.
3. **Gap Analysis**: Detected skills are mapped against predefined job core requirements (utilizing semantic matching and synonym resolution, like mapping "MySQL" to "SQL", or satisfying a "programming language" requirement when matching one language).
4. **Output**: Match percentage and missing skills are prioritized and presented logically. 
5. **Guidance**: A weekly personalized learning roadmap outlines the missing skills, prioritizing foundational high-priority competencies first.

## Quick Start
These tools are intended to test if the codebase safely initializes directly on your developer workstation. 

### Prerequisites
* Python 3.10+
* Groq API Key (get from https://console.groq.com)

### Run Commands
```bash
git clone https://github.com/ashishSoni1234/skill-bridge-career-navigator.git
cd skill-bridge-career-navigator/skill_bridge
pip install -r requirements.txt

cp .env.example .env
# Edit .env and supply your GROQ_API_KEY

streamlit run app.py
```

### Test Commands
Ensure all background workflows, fallbacks, and gap analysis algorithms work natively on your machine by running the test suite via `pytest`.
```bash
python -m pytest tests/test_app.py -v
```

## AI Disclosure
* **Used AI tools:** Yes (ChatGPT, LLaMA 3.1 / LLM APIs via Groq)
* **Verification:** Manual code review, rigorous whitelist filtering on output, and comprehensive unit testing.
* **Rejected Example:** Raw AI extraction sometimes provided descriptive sentences or email addresses. We wrote aggressive post-processing normalization (rules) to extract only valid string data points filtering out invalid noise.

## Tradeoffs & Prioritization
* **Heuristics Over Custom NLP Models:** Rather than training a custom NER (Named Entity Recognition) model for classifying experience and skills locally, we integrated LLaMA 3.1 backed by strict heuristics (fallback mapping and whitelist). This favored development speed and lower architectural complexity.
* **Synthetic Local Dataset vs Live Scraping:** Opted to hard-code a robust 20-role core competency list in `dataset.json` rather than scraping live jobs from LinkedIn/Glassdoor. This sidestepped external API limits, ensuring the system can be cloned and run fully independently. 
* **Monolithic Structure:** Developed a streamlined monolithic application via Streamlit instead of breaking out a standalone Python/FastAPI backend and React frontend. This expedited delivery of interactive gap analysis visualization without boilerplate code delays.
